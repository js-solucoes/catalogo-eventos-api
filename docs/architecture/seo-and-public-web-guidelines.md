# SEO and public web guidelines

## Quando considerar SEO como requisito crítico
- sites institucionais
- páginas públicas de marca
- landing pages
- catálogos públicos
- páginas de aquisição e descoberta orgânica

## Diretrizes
- considerar renderização adequada para indexação
- definir estratégia de metadata por página
- garantir headings e semântica corretos
- otimizar performance web
- considerar sitemap, robots e canonical quando necessário
- instrumentar tag managers com governança mínima
- evitar scripts excessivos que degradem experiência e indexação

## Avaliação arquitetural
Quando SEO for prioritário, avaliar se a solução deve favorecer SSR, SSG ou estratégia híbrida.
Essa decisão deve considerar:
- volume de conteúdo
- frequência de atualização
- necessidade de personalização
- performance
- operação
